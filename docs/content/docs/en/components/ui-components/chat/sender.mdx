# ChatSender

Message input and send component, supports text input, file upload, voice recording.

## Usage in Vue

```vue
<template>
  <ChatSender
    :is-stream-load="isLoading"
    @send="handleSend"
    @stop="handleStop"
  />
</template>
```

## Usage in HTML/React or Other Frameworks

```html
<div id="containerId"></div>
<script>
ADPChatComponent.ChatSender.mount('#containerId', {
  isStreamLoad: false,
  onSend: (value, fileList) => {},
  onStop: () => {}
})
</script>
```
